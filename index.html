Political Blog Topic Analysis (STM)

Overview

This project is mainly conducted as personal study of Structural Topic Modeling applications in exploring topics and their relationship to metadata. By applying different features in the stm package, we can process raw text data, analyze it and visualize the findings with the plotting tools. Basically, Implementation of document metadata is what makes structural topic model different from the traditional Latent Dirichlet Allocation (LDA) and Correlated Topic Model (CTM) models. Metadata covariates can be either functioned as topical prevalence, which tells us how much each topic is discussed within a document, or functioned as topical content, which refers to the word rate use within a topic.(This study is adapted from the "article" stm: R Package for Structural Topic Models")

In this project, the CMU 2008 US political blog posts are analysed using structural topic modeling method to discover different topics and their relationships to political affiliations.

Applied Coding Languages & Technical Skills: R Language, Natural Language Processing, Topic Modeling, Word Cloud, Data Processing, Data Visualization, stm package
